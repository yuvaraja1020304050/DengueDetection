About the System

The Dengue Prediction System uses advanced machine learning algorithms to predict the likelihood of dengue infection based on various factors like age, gender, NSI levels, IgG levels, and environmental factors.

Our system employs three state-of-the-art models:

  • XGBoost - Our highest accuracy model
  • Random Forest - Excellent feature importance analysis
  • Support Vector Machine (SVM) - Great for binary classification
